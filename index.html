Quick Reference — Which GPT for What
Starting a new listing
→ Listing Launch Method™ Playbook
Download the playbook first. It maps every phase from intake call to MLS Active — with the exact AI tool to use at each step.
Writing a listing
→ Listing Domination GPT
Give it the address, features, target buyer, and price. Ask for MLS + social + brochure versions.
Listing appointment prep
→ Seller Psychology Engine + Simulator
Run a practice session first. Then generate scarcity + timing scripts for the real appointment.
Buyer objection
→ Conversion GPT
Tell it the objection, client type, and medium. Get a word-for-word script instantly.
Social media post
→ Brand Voice GPT (Phase 3)
Tell it the platform, goal, and topic. On-brand GoldLine content with the right hashtags.
Price reduction conversation
→ Seller Psychology Engine
Use the loss aversion trigger. Provide days on market, current price, and target price.
Open house follow-up
→ Conversion GPT + Open House Playbook
Classify visitors by temperature, then generate personalized same-day follow-ups.
